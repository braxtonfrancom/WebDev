Software Development Plan

Phase 0: Requirements Specification

My goal is to build a fully functional calculator using Javascript and Vanilla JS;).

I haven't worked with Javascript at all but I am actually really excited to become more familiar with it because it is so common. I think that it will be really cool to convert between HTML and Javascript. Some challenges that I forsee would be getting caught up in the strange syntax of Javascript. I'm hoping that I can learn Javascript with an open mind and not rely too much on my knowledge of other languages.

Phase 1: System Analysis

I will be taking in two number inputs and an operation. I will need to limit the input to numbers, otherwise I will run into errors trying to perform operations on said inputs. The operation will be selected by the user from a dropdown list of predefined operations. I will then evaluate the mathematical expression.

My output will be a fully functioning Javascript calculator that will generate the correct output given the user's input values and operation selection.

There really won't be any algorithms or formulae used in this program. Just good ol' HTML, CSS, and Javascript.

Phase 2: Design

  • All of my work will be done in script.js. I will not modify the index.html page.
  • script.py
    • function calculator()

      In this function I will define all HTML elements and assign them their correct names, id's attributes, and CSS styling.

      I will also make a function call to append each time I want to create a new div.

    • function append()

      In this function I will create a new date object. I will then verify that both of the inout fields arent blank. If they are not black I will insert a new div that contains the date object and the equation to calculate.

      If one or both of the fields is empty, I will set the background color to red, set the textContent of the div to an error message and then insert the new div.

      Lastly, I will put an event listener on every new div created. This will make it so that when any of the divs created is clicked, it will be deleted.

Phase 3: Implementation

I implemented all of the above code and more, as well as made some slight modifications to the CSS starter code to create a functional Javascript calculator.

I learned that Javascript is a very powerful tool. I also learned that it has some pretty unique and sometimes funky syntax. It can be quite tricky at times figuring those things out but this project helped me to develop a lot more confidence in my ability to use Javascript.
