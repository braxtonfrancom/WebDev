Check these boxes out!

  • Checkbox buttons are created with the <input> element with the type attribute set to checkbox
  • An individual checkbox is tied to a label by its id attribute
  • A checkbox is identified by the server through its value attribute
  • A group of related checkboxes share the same name attribute
  • You can pre-select a checkbox with the checked attribute

  • Radio buttons are created with the <input> element with the type attribute set to radio
  • An individual radio button is tied to a label by its id attribute
  • A radio button is identified by the server through its value attribute
  • A group of related radio buttons share the same name attribute
  • Only one radio button in the group may be selected at a time
  • You can pre-select a radio button with the checked attribute
