Numeric inputs

  • These <input> elements are constrained to accept only numeric data
  • Note: this validation is easily defeated; consider it a convenience, nothing more.
  • Some browsers add UI features to make entering numbers more convenient (e.g. spinners, numeric keyboard)

Miscellaneous inputs

  • A color picker widget can be summoned by setting an <input> element's type to "color"
    • Its value is expressed as a hexidecimal triplet #RRGGBB
  • A file picker widget is summoned with type="file"
