0. HTML Elements

<html>

  • Name: html
  • Purpose: This is the HTML root element. It must be unique on every document
  • Display: N/A

<code>

  • Name: code
  • Purpose: Displays it's contents in a monospace font that resembles computer code
  • Display: Inline

<img>

  • Name: img
  • Purpose: Embeds an image into the document
  • Display: Inline(by default)

1. HTML Character Entities

&lt;

  • Name: < (less-than)
  • Purpose: Expresses a relationship between two objects and states that the first object is less than the second
  • Entity Number: &#60

&gt;

  • Name: > (greater-than)
  • Purpose: Expresses a relationship between two objects and states that the first object is greater than the second
  • Entity Number: &#62

&amp;

  • Name: & (ampersand)
  • Purpose: Represents 'and'
  • Entity Number: &#38

2. CSS Selectors

element

  • Name: element
  • HTML feature this selector finds: Elements
  • Example: this could select all <p> elements

.class

  • Name: .class
  • HTML feature this selector finds: Classes
  • Example: .intro selects all elements with class="intro"

#id

  • Name: #id
  • HTML feature this selector finds: ID's
  • Example: #mystyle selects the element with id="mystyle"

:nth-child

  • Name: nth-child
  • HTML feature this selector finds: The child of whatever element is typed before ':nth-child'
  • Example: 'p:nth-child(2)' selects every <p> element that is the second child of its parent

3. CSS Properties

background-color:

  • Name: background-color:
  • Purpose: Allows you to change the color of an element's background
  • Possible values to take: background-color:purple, background-color:lightskyblue

font-style:

  • Name: font-style
  • Purpose: Defines the font style for the text
  • Possible values to take: font-style:italic, font-style:normal

font-size:

  • Name: font-size
  • Purpose: Defines the font size for the text
  • Possible values to take: font-size:3rem, font-size:25px
